# MegaJump

A simple internal cheat/mod for MegaBonk.

Features:

F2 - Increase Attack Speed by 1x

F3 - Toggle ESP

F4 - Turn ESP from Squares -> Path Lines

F5 - Heal the player

F6 - Pickup All XP on Map

F7 - Adds Jumps (I think there's fall damage?)
